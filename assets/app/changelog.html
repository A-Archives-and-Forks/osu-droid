[insert version here] ================== # Additions - Add beatmap attribute display by [Rian8337](https://github.com/Rian8337) - Can be displayed by pressing the text with difficulty values (CS, AR, OD, HP, star rating) in song select. - Add option for selecting language in settings by [Rian8337](https://github.com/Rian8337) - Can be accessed under the `General` category in settings. - Changing the language requires restarting the game. Therefore, it is disabled in multiplayer. # Changes - Do not update OD value when using Precise mod in song select by [Rian8337](https://github.com/Rian8337) - It's the hit window that is different, not the OD, so attempting to "pair" OD like this is conceptually wrong. - Rework input handling in gameplay by [Rian8337](https://github.com/Rian8337) - This solves an issue where inputs between two frames (commonly known as subframe inputs) are discarded except the last input before the next frame (an input is either a finger press, finger drag, or finger lift). - Makes hitobject judgements "consume" finger presses sequentially. This means that if there are `n` hitobjects with the same starting time, and they overlap, they would require `n` presses to hit instead of one (there is a [beatmap](https://osu.ppy.sh/beatmapsets/1157609#osu/2439789) that abuses this behavior). # Bug Fixes - Fix potential crash when canceling gameplay loading while slider paths are being loaded by [Rian8337](https://github.com/Rian8337) - Fix non-host players being able to change required mods' settings in free mod by [Rian8337](https://github.com/Rian8337) - Fix score overflow when playing a beatmap with information that has not been fully populated by [Rian8337](https://github.com/Rian8337) - Full population happens in background difficulty calculation. This bug happens when a player plays a beatmap with the ScoreV2 mod enabled and with the beatmap not being calculated yet. - Fix replay import notifications not showing by [Rian8337](https://github.com/Rian8337) - Fix Really Easy mod approach rate adjustment only counting Custom Speed mod instead of all constant rate-changing mods by [Rian8337](https://github.com/Rian8337) - Fix wrong displayed score when opening it from pp-based beatmap leaderboard if the score is different from the score-based leaderboard by [Rian8337](https://github.com/Rian8337) # Removals - Remove `Remove slider lock and spinner lock` setting - With the rework to gameplay input handling, this is enabled by default and thus is no longer needed.
